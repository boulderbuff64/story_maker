# Skill: Financial Analysis & Pro Forma

## When to Use
When asked to create, update, or verify financial projections for the property.

## Data Sources
- Property data: `docs/reference/property_data.md`
- Comparable sales: Provided by owner or researched
- Market rents: Point Loma / 92107 rental data
